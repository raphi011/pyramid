# Layout Patterns

Mobile-first responsive layout with bottom navigation on mobile and sidebar on desktop.

## App Shell

```
Mobile:                          Desktop (lg+):
┌──────────────────┐            ┌────────┬─────────────────┐
│   Page Content   │            │Sidebar │  Page Content    │
│                  │            │        │                  │
│                  │            │ Nav    │                  │
│                  │            │ items  │                  │
│                  │            │        │                  │
├──────────────────┤            │        │                  │
│ BottomNav (fixed)│            │        │                  │
└──────────────────┘            └────────┴─────────────────┘
```

## Navigation Items

| Icon | Label | Route |
|------|-------|-------|
| Trophy | Rangliste | `/` |
| Bell | Neuigkeiten | `/neuigkeiten` |
| User | Profil | `/profile` |
| Settings (sidebar only) | Einstellungen | `/settings` |

## Page Layout

Every page is wrapped in `PageLayout`:

```tsx
<PageLayout
  title="Rangliste"
  subtitle="Saison 2026 · TC Musterstadt"
  action={<SeasonSelector />}
>
  {children}
</PageLayout>
```

## Responsive Breakpoints

| Breakpoint | Width | Layout |
|------------|-------|--------|
| Default | < 1024px | Bottom nav, full-width cards, `px-4` |
| `lg` | ≥ 1024px | Sidebar nav, `max-w-2xl` content, `px-6` |

## Card Patterns

Cards are the universal container. Use `CardHeader` + `CardContent` for structured content.

```tsx
<Card>
  <CardHeader>
    <CardTitle>Section Title</CardTitle>
    <CardAction><Button size="sm">Edit</Button></CardAction>
  </CardHeader>
  <CardContent>
    {/* content */}
  </CardContent>
</Card>
```

## Consistency Checklist

- **Icons**: Heroicons 2 only, `size-5` for UI, `size-6` for nav
- **Spacing**: `gap-6` between sections
- **Text hierarchy**: max 3 levels per view (heading + body + caption)
- **Loading**: every data view shows `Skeleton` while loading
- **Empty**: every list handles zero items with `EmptyState`
- **German text**: all user-facing labels in German
